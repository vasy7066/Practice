Инструкция для Windows:

1) Откройте терминал и перейдите в корневую директорию проекта
2) Введите команду pip install auto-py-to-exe для установки необходимой библиотеки
3) Введите команду pip install -U pyinstaller для установки необходимой библиотеки
4) Введите pip install -r requirements.txt для установки необходимых библиотек
5) введите pyinstaller 1.py
6) Перейдите в папку с exe файлом, введя команду: cd .\dist\1
7) Введите команду start 1.exe для запуска программы

Инструкция для Linux:

1) Необходимо установить Anaconda с официального сайта
2) Запустите файл install.sh
3) После окончания создания виртуальной среды и установки всех необходимых зависимостей запустите файл start.sh, в дальнейшем для запуска программы используйте этот же файл
